# 2학기 캡스톤 디자인 수업 - SSS

프톤트 배포: https://dlsss.netlify.app/   
API 서버: https://dldm.kr/swagger-ui/index.html

# 프로젝트 소개
SSS는 축구 4대 리그 기록을 볼 수 있는 웹 사이트입니다. 축구 데이터는 rapidApi-API-FOOTBALL에서 가져왔습니다.  
이 데이터를 통해 선수 프로필, 팀 전적, 랭킹을 조회하는 기능을 만들었습니다.
